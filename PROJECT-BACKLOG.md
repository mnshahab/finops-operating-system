# FinOps OS Project Backlog

## Purpose

This backlog tracks all deliverables required to build and maintain the FinOps OS repository. It provides visibility into project status, priorities, ownership, and upcoming work.

---

## Project Status Legend

| Status      | Description                     |
| ----------- | ------------------------------- |
| Not Started | Work has not begun              |
| In Progress | Currently being developed       |
| Review      | Awaiting validation or approval |
| Complete    | Finished and published          |
| Future      | Planned for a later release     |

* AI FinOps framework is published
