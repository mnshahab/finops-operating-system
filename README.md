# FinOps Operating System

An open-source operating framework for FinOps teams.

This repository provides practical templates, playbooks, KPIs, governance models, reporting structures, and operational runbooks for running FinOps across AWS, Azure, GCP, Kubernetes, SaaS, data platforms, and AI workloads.

## Purpose

FinOps teams need more than concepts. They need repeatable operating practices.

This repository helps teams:

- Build a FinOps operating model
- Define governance and accountability
- Improve cloud cost visibility
- Create showback and chargeback models
- Track KPIs and business value
- Manage commitments and optimization
- Govern AI and GPU spend
- Run executive cost reviews
- Standardize anomaly response
- Mature from crawl to walk to run
