# Contributing

Thank you for contributing to FinOps Operating System.

## Contribution Areas

You can contribute:

- FinOps playbooks
- KPI definitions
- Dashboard ideas
- SQL queries
- Cloud optimization examples
- AI FinOps practices
- Kubernetes cost management patterns
- Governance templates

## Standards

All contributions should be:

- Practical
- Clear
- Vendor-neutral where possible
- Safe to share publicly
- Free of company confidential data

## Pull Request Checklist

Before submitting:

- Remove customer names
- Remove account IDs
- Remove pricing from private contracts
- Use generic examples
- Keep language simple
